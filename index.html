Architecture

v0 stack

  • shellElectron + Vite + React — shared with Crisp / yawp
  • terminalxterm.js + node-pty — a real shell; type claude
  • previewembedded Vite dev server — compiles & hot-reloads the focus file
  • pickeropen-folder + ⌘P palette — choose project & focus component
  • wrapperper-project preview wrapper — provides styles/providers (manual now, sniff later)

The preview wrapper

A naked component won't render without the project's styling. Set this once per project:

// .dogfood/wrapper.tsx — set once per project
import '../src/styles/globals.css'
import { Providers } from '../src/app/providers'
import Focus from './focus'   // whatever you're building

export default function Preview() {
  return <Providers><Focus /></Providers>
}
