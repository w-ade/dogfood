Dogfood

A tiny desktop forge for building interface with Claude Code — a translucent, Cursor-style workspace where you prompt, build, and preview UI pieces inside the real repo.

Getting started

  • Open a project folder with ⌘O — Dogfood works against your real repo, not a sandbox.
  • Open the terminal with ⌘J and type claude to start Claude Code.
  • Prompt it to build a component; it edits files in place, where they belong.

The interface

Dogfood is laid out like a focused code editor, with a translucent canvas:

  • Title bar — the app name, a light/dark toggle, and settings, kept thin like a native macOS window.
  • Canvas — the calm, gridded stage where your component renders.
  • Terminal panel — a bottom panel of real shells in tabs; drag its top edge to resize, or hide it with the chevron.
  • Status bar — a slim footer with quick state.

Terminal

The terminal is a real shell (xterm + a pty), not a chat box. Open multiple as tabs with +, switch between them, and each keeps its own live session. Type claude in any of them.

Themes

Toggle light and dark with the sun/moon in the title bar. Dark is a frosted, translucent glass; light is clean and bright. The grid stays in both.

Keyboard shortcuts

  • ⌘J  Hide / show the terminal
  • ⌘P  Search files
  • ⇧⌘B  Open browser
